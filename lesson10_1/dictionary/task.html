Задача "Словарь"

Дан словарь vocab. Нужно реализовать приложение, которое выполняет поиск по ключу в этом словаре.

Напишите представление, которое берет из урла текстовый сегмент, ищет по полученному значению сегмента ключ в словаре vocab и возвращает Значение для ключа <значение переданного ключа> - <найденное по этому ключу значение в словаре>.

Если такого ключа нет в словаре, то возвращает строку "Ключ не найден" и код ответа 404.

Например, по урлу /cat/ приложение ответит "Значение для ключа cat - кошка".
